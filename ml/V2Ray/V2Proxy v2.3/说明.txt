#**************使用说明*****************

1. 解读setting.ini设置
    IP         : 服务器IP   如果是腾讯云服务器(封Host) 请注意流量~~
    Port       : 端口
    UUID       : 服务器端设置的UUID
    alterld    : 要求与服务器端配置一致 官方推荐设置为: 32
    security : 加密协议 推荐使用"aes-128-cfb"  其他加密协议: "auto" "none" "aes-128-gcm" "chacha20-poly1305"
    Path       : 首头混淆
    Host       : Host混淆
    XH         : 只需要修改混淆
    Dns        : 国外IP推荐使用"8.8.8.8"
    
2. V2Ray写法
   首头留空     : '\/'
   首头混淆     : 'http://混淆/'
   XH头混淆    : '"X-Online-Host": ["混淆"],'
 
3. 联通电信互联网卡使用方法:
    以腾讯大王卡为例: 
    #TCP放行  流量直连
    TCP_FX=''

    #UDP放行  游戏直连
     UDP_FX=''

4. 高级设置
    #UDP代理     ON=开启 (通过TCP转发UDP数据)    OFF=关闭 (本地4G直连 消耗运营商流量)   
    UDP='ON'

    #WIFI代理     ON=开启 ( WIFI通过V2Ray代理)      OFF=关闭 
     WIFI='ON'

    #热点代理     ON=开启 (热点共享上网)                  OFF=关闭
     Hot='ON'

    #网络开/关    ON=开启 (数据开关 刷新网络)        OFF=关闭
     WL='OFF'
     
     
    #网络类型     可选值: tcp/ws/kcp
     Network=tcp


    #破版权         ON=开启(国外服务器)                OFF=关闭(国内服务器)
    Adblock_Bvc='ON'
  
    破版权域名来自妖友千影
    
5. 实验室功能
    #UDP禁网
    UDP_JW=''

     #网卡放行            联发科=ap+ 小米=softap+ VPN=tun+ 
     NC_FX=''

     #内网白名单        前两段IP        比如10.20
     IP_BMD=''

     #XH混淆               只需更改混淆中的box.10155.com    
     XH='"X-Online-Host": ["box.10155.com"],'
     
5.  腾讯大王卡用户:
     比如腾讯视频UID=10023 视频完全放行 其他应用通过V2Ray代理
     腾讯视频有广告会触发日租包、造成的跳点不负责哦(当然你有好莱坞会员就另说了)
     配置如下:
     TCP_FX='10023'                 #TCP放行腾讯视频 流量不通过V2Ray代理

     比如王者荣耀UID=10086 游戏完全放行、其余应用通过TCP转发UDP数据
     配置如下：
     TCP_FX='10086'                 #TCP放行王者荣耀 流量不通过V2Ray代理
     UDP_FX='10086'                #UDP放行王者荣耀  流量不通过V2Ray代理

6. 问题解疑

1. 为什么本机开启热点、对方连接热点却一直获取IP中？
答: 本机开热点需放行 "wlan+" 网卡、否则对方无法连接本机热点

2. 看爱奇艺、优酷、哔哩哔哩等视频软件无法播放？
答: 请更换Dns试试或仔细查看配置是否有误

3. 观看腾讯视频产生的跳点忽高忽低？
答: 首头混淆不支持腾讯视频、导致不免或高跳（具体以地区为准）

4. 以北京电信为标准:

应用名         使用量   跳点     协议            方式
应用商店     200m     0.5m    TCP            下载
腾讯视频     100m     0.2m    TCP            视频
YY直播          60m      0.5m    UDP            直播
王者荣耀      10m       0.4m    TCP/UDP

5. 联通使用跳点不稳定？
答: V2Ray对电信友好、对个别地区联通友好、酌情使用！
